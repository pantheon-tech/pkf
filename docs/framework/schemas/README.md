# Reference Schemas

> JSON Schema definitions for PKF document validation.

## Overview

This directory contains reference copies of the JSON schemas used for PKF validation. The authoritative schemas are in the root `/schemas/` directory.

## Available Schemas

| Schema | Purpose |
|--------|---------|
| `changelog-entry.schema.json` | Changelog entry validation |
| `directory-structure.schema.json` | Directory structure validation |
| `document-frontmatter.schema.json` | Generic document frontmatter |
| `issue-item.schema.json` | Issue register entries |
| `pkf-config.schema.json` | PKF configuration file |
| `register-entry.schema.json` | Base register entry |
| `template-metadata.schema.json` | Template placeholder metadata |
| `todo-item.schema.json` | TODO register entries |

## Usage

These schemas are referenced by the PKF validation tooling to ensure documents conform to expected formats.
