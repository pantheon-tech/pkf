# Reference Templates

> Document templates for PKF-compliant documentation.

## Overview

This directory contains reference copies of PKF document templates. The authoritative templates are in the root `/templates/` directory.

## Usage

Templates provide consistent starting points for new documents. Each template includes:
- YAML frontmatter with required fields
- Standard document structure
- Placeholder variables for customization

See the [PKF Specification](../specifications/PKF-SPECIFICATION.md) for template usage details.
